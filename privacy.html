Privacy Policy

Effective Date: July 23, 2025

1. No Personal Data Collection

The Bot does not collect personal data such as names, emails, or passwords.

2. Functional Data Storage

To operate properly, the Bot stores:

  • User IDs
  • Server (guild) IDs
  • Data that users manually input into the Bot (e.g., character names, registration info)

3. Data Removal

You may remove your data:

  • By using the reset command (if available), or
  • By contacting the Bot owner to request deletion

4. Data Usage

Stored data is only used for the purpose of making the Bot work as intended. Data is not shared with any third parties.

5. Changes to This Policy

The policy may change over time. Changes will be reflected here, and continued use implies acceptance of those changes.
